# Channel Architecture

## Channels

- referral_partner
- strategic_partner
- reseller
- affiliate

## Core Rules

- exclusive territory offers are partner-only
- standardized offers can be affiliate-enabled
- directories prioritize partner monetization
- authority sites prioritize affiliate monetization

## Placement Rules

- authority pages → affiliate allowed
- directory lead pages → affiliate restricted
- post-lead → hybrid allowed

## Offer Types

- territory nodes (partner only)
- SaaS systems (affiliate allowed)
- authority memberships (affiliate allowed)

## Enforcement Principle

Channel rules must be enforced via config, not manual decisions.
