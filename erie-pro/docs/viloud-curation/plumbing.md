## Erie-specific filter

Prefer videos that mention or visibly work in northern/cold climates. Erie averages 60-80 freeze-thaw cycles per winter — Florida plumbing content is fine but feels less relatable. Bonus points for videos that mention winterization, frozen pipe repair, or basement plumbing concerns (most Erie homes have basements; many southern-US channels don't).

When this channel is curated well, the following niches can largely reuse the same channel (or duplicate it with different branding):

- `drain-cleaning`
- `sewer-line-repair`
- `water-heater-services`

For Phase 1, you can either:
- Point all 4 niches to the same Viloud channel (saves setup time; same content)
- Create 4 separate channels with overlapping content (better SEO uniqueness; more setup work)

Start with shared channel for Phase 1, split later if data shows distinct audience behavior.
